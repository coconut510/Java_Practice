@Switch 문
-> 변수의 값을 case 에서 찾아 명령을 수행하는 조건문
-. 해당하는 값이 없을 경우에는 default 의 명령문을 수앻

* case와 명령문 사이에는 콜롬(:)을 해야함

switch(변수)
{
	case 1 : 실행문 1;  // 변수 안에 있는 값이 1일때
		break;

	case 2 : 실행문 2;	// 변수 안에 있는 값이 2일때.
		break;

	defualt : 실행문 3,	// 변수 안에 값이 mcase 문에 없을 경우에 실행(else 와 같음)

}



@while

-> 사용자가 원하는 만큼 명령들을 여러번 실행하는 제어 구조
-> 특정 조건이 만족되는 동안 명령을 계속적으로 실행함
-> while 문의 조건이 참일 경우에만 루프 처리를 반복함
* 반복문을 처음 접할땐 초기, 조건, 증강 값을 생각하며 접근하면 덜 어려움

### 표현식 ###

while(조건문)
{
	실행문;
}
